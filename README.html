The Comet Way Agent Kernel

Thank you for downloading this release of the Comet Way Agent Kernel.

We have overwhelmingly found that agent-based application design is an effective pattern for the creation of any kind of software. This includes autonomous server applications, web-based applications, embedded software, and interactive GUIs. Virtually any type of application can be written using the Comet Way Agent Kernel.

The Facts

  • The Agent Kernel implements a dynamic component software model specifically designed for deploying Agent Applications.
  • Link up interchangeable agent-based components to create robust Agent Applications.
  • Includes agents for handling operations between web-based (HTTP and HTTPS), email-based (SMTP and POP3), database (JDBC and XML), and GUI-based (Swing) services.
  • Designed for the command line; easily enhanced by your favorite IDE.
  • A mature framework for Agent Programming.
  • Write your own customized agents using the Java Programming Language.
  • Created especially for the Java programming language from Sun Microsystems.
  • The Java Programming Language ensures 100% compatibility with a growing number of 3rd party Java APIs and platforms.
  • Complete details and documentation on our website at www.agentkernel.com

Compatibility

This release of the Comet Way Agent Kernel is implemented entirely in Java but it uses Java functionality found only in Java versions 1.4 and greater. For best results recompile under your preferred Java platform before using.

The Comet Way Agent Kernel has been used extensively with the following popular environments:

  • Sun Microsystems Java Developer Kit (J2SE)
  • J2SE for Apple Mac OS X

Release Notes - Version 2.0

Comet Way is proud to release Version 2.0 of the Comet Way Agent Kernel, a release containing many upgrades, and advances which promise a stable and fruitful path for developers using the platform.
Here are the highlights:
  • new lightweight API for streamed generation and parsing of XML documents
  • new AgentController model provides a state driven interface to agents
  • new Agent service manager methods for improved Agent connectivity
  • new ClassFactory API for the management of agent class loading
  • new 'ak' command option -reporter specifies the default reporter class
  • new 'ak' command arguments agent classnames accepted in addition to .agent filenames
  • new JDBC agents and examples for easy database access
  • improved javadocs and documentation
  • new "biolife" agent simulation, originally presented at SummerCon 2001 in Amsterdam
  • new "robot" agent demo
  • improved build scripts
  • jakarta-oro-2.0.5 included for regular expression
  • many obsolete classes removed
  • many kinetoscope classes integrated into com.cometway packages
